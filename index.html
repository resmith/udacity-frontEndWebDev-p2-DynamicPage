Async Await

Await is part of ES8 / 2017 and it provides a cleaner method for consumption of promises. It is not used for creating promises

Format

The function containing the async process is preceded with the word "async" - The async process being called should be 1) in the async function and 2) preceded with the word "await" - If a post process is needed after the async process is completed, then a call back should be provided

javascript async function asyncFunctionName(arguments) {
const retrievedData = await functionToCall(arguments);
return retrievedData; }
asyncFunctionName(valuesToPass).then(valuesReturned => functionToRun(valuesReturned));
